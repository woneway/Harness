# 股票分析 Pipeline 设计文档

**日期：** 2026-03-21
**目标：** 对单只 A 股给出买/卖/持有信号，输出终端摘要 + Markdown 报告

---

## 概述

使用 Harness pipeline 对单只 A 股（沪深）进行综合分析：技术面 + 基本面 + 新闻情绪，最终给出可执行的买/卖/持有裁决。

## Pipeline 架构

```
股票代码 (如 "600036")
    │
    ▼
FunctionTask [数据获取]
  • akshare 获取近 120 天日K线（开/高/低/收/量）
  • akshare 获取最新基本面（PE、PB、ROE、营收增速）
    │
    ▼
FunctionTask [技术指标计算]
  • MA5 / MA20 / MA60
  • MACD (DIF / DEA / 柱状值)
  • RSI(14)
  • 布林带（上轨/中轨/下轨）
    │
    ▼
LLMTask [技术分析]
  • 分析指标数据
  • 输出技术信号：看涨 / 看跌 / 中性
  • 关键理由（突破/死叉/超买超卖等）
    │
    ▼
LLMTask [基本面分析]
  • 分析 PE、PB、ROE、营收增速
  • 与行业均值对比
  • 输出估值信号：低估 / 合理 / 高估
    │
    ▼
LLMTask [新闻情绪分析]
  • WebSearch 搜索近期新闻（公司 + 行业）
  • 输出情绪信号：积极 / 中性 / 消极
  • 关键事件摘要
    │
    ▼
LLMTask [综合裁决]
  • 整合三维信号（技术 + 基本面 + 情绪）
  • 结论：买入 / 持有 / 卖出
  • 信心评分（1-5）
  • 核心理由（3条）
  • 风险提示
    │
    ▼
FunctionTask [输出]
  • 打印终端摘要（结论 + 信心 + 核心理由）
  • 保存完整报告到 <股票代码>_analysis.md
```

## 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| 数据源 | akshare | 免费、A股覆盖完整 |
| 技术指标 | pandas-ta | 纯 Python，依赖少 |
| LLM 调用 | Harness LLMTask | 框架标准方式 |
| 报告格式 | Markdown | 易读、可版本管理 |

## 输入 / 输出

**输入：**
```bash
python examples/stock_analysis.py 600036           # 招商银行
python examples/stock_analysis.py 000858           # 五粮液
python examples/stock_analysis.py 600036 --output ./reports
```

**终端输出（摘要）：**
```
==============================
  股票：招商银行 (600036)
  结论：持有
  信心：3/5
  核心理由：
    1. 技术面：MA20 支撑有效，MACD 金叉形成
    2. 基本面：PE 6.2x 低于行业均值，股息率 5.1%
    3. 情绪：近期无重大负面事件
  风险提示：宏观利率政策变化、房地产敞口
==============================
```

**报告文件：** `600036_analysis.md`（包含完整三维分析 + 数据表格）

## 关键设计决策

1. **FunctionTask 做数据获取和计算**：保持 LLM 专注于分析，不让 LLM 执行代码
2. **三个独立 LLMTask 分析**：各维度独立分析后综合，比单次长 prompt 质量更高
3. **综合裁决 LLMTask 拿到前序全部结果**：通过 `results[0..4]` 访问所有上下文
4. **命令行参数**：支持股票代码 + 可选输出目录，与已有 examples 风格一致
